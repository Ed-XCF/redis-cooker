redis-cooker
=
An redis operation proxy package
-

Python Version Support
^^^^^^^^^^^^^^^^^^^^^^

redis-cooker supports Python 3.6.

Installation
-
To install redis-cooker, simply:

.. code-block:: bash

    $ pip install xxxxx

or from source:

.. code-block:: bash

    $ python setup.py install
Getting Started
---------------

.. code-block:: pycon

    >>> import xxxxx

By default, all responses are returned as `str`.
